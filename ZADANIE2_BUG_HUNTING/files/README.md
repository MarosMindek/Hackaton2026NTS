# KnihaPlus

Interný systém správy knižničného fondu.

## Spustenie

```bash
python app.py
```

## Funkcie

- Správa kníh (pridanie, vyhľadávanie)
- Správa členov
- Výpožičky a vrátenia
- Štatistiky

## Poznámky

Systém bol napísaný rýchlo, môžu sa vyskytnúť problémy. Dokumentácia nie je kompletná.
