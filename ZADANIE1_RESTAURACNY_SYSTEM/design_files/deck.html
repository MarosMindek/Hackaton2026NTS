Restaurant Module · Overview02 / 17
How we'll judge the work

Five principles.

01

Clarity over density

Progressive disclosure beats dense panels. Surface only what the user needs at this step.

02

Accessibility first

WCAG 2.1 AA — contrast, hit targets, screen reader, assisted dining as a feature.

03

Consistent across surfaces

Phone, in-room tablet, kiosk & console all speak the same visual language.

04

Graceful degradation

PMS, KDS or payments offline? The module says so and keeps the floor moving.

05

Staff efficiency

The console is a working tool. Density, keyboard flow and quick-access patterns.
